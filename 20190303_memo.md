# Const/Variable/Immutable/Mutableについて
* Const : 定数
* Variable : 変数
  * Immutable : 不変
  * Mutable : 可変
## KotlinのConst/Variable/Immutable/Mutableについて 
* Const : companion object (Const val)
* Variable : 
  * Immutable : val
  * Mutable : var
  
### Kotlinの言語仕様
* Variable : 
  * Immutable : val
    * Const : companion object(const val)
  * Mutable : var

## JavaのConst/Variable/Immutable/Mutableについて
* Variable : 
  * Immutable : final
    * Const : static final
  * Mutable : 普通の変数
